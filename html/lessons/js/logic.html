Arrays

Los arrays son colección de elementos, se crean con corchetes. Los arrays son objetos.

  • Arrays

    const myArray = [];          // Array literal
    const myArray = new Array(); // Objeto array

  • Modicando el array

    const heroes[0]= 'Wonder Woman'; // Agrega en el primer elemento
    const heroes[1]= 'Aquaman';      // Agrega en el segundo elemento
    const heroes[2]= 'Flash';      
    const heroes[3]= 'Cyborg';     
    const heroes[4]= 'Batman';     
    heroes;
    const avengers = ['Captain America', 'Iron Man', 'Thor', 'Hulk'];
    avengers;
    delete avengers[3];  // Remueve el elemento pero lo deja vacío
    avengers;            // El último elemento ahora esta vació

  • Reestructurar arrays

    const [x,y] = [1,2]; // Reestructuración permite asignar valores al mismo tiempo
    [x,y] = [1,2];       // Cada variable se asigna fuera del array

  • Propiedades y métodos

    avengers.length;              // Muestra cuantos elementos tiene un array
    avengers[avengers.length -1]; // Muestra el último elemento
    avengers.push('Vision');      // Agrega un elemento al final del array
    avengers.unshift('Ultron');   // Agrega un elemento al inicio del array
    avengers.pop();               // Remueve el último elemento del array
    avengers.shift();             // Remueve el primer elemento del array
    avengers.concat(['Black Widow', 'Scarlet Witch']); // Crea un subarray uniendolo otro
    avengers.join(' & ');         // Convierte un array en un string
    avengers.slice(2, 4);         // Crea un subarray a partir de un rango de elementos
    avengers.splice(5, 0, 'Falcon');   // Inserta un elemento
    avengers.splice(3, 1, 'Loki');     // Remueve un elemento e inserta otro
    avengers.reverse();           // Invierte el orden del array
    avengers.filter(function);    // Filtra el array en base a una función
    avengers.sort();              // Ordena el array alfabeticamente
    avengers.includes('Hulk', 1);     // Busca si existe a partir de un index

  • Arrays multidimensionales

    const coordinates = [[1,3], [4,2]];    // Un arrays de arrays
    coordinates [1] [0];                   // Accede al primer valor del segundo array
    const summer = ['Jun','Jul','Aug'];   
    const winter = ['Dec','Jan','Feb'];   
    const nested = [ summer, winter];      // Une los dos arrays en un array multidimensional
    const flat = [ ...summer, ...winter ]; // Une los dos arrays en un array del mismo nivel

    Un arrays de arrays es un array multidimensional
