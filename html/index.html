FrontEnd desde cero

Desarrollador FrontEnd

Un desarrollador front-end diseña y/o desarrolla sitios web, aplicaciones web, o aplicaciones nativas ejecutando tecnologías web.

Un equipo de desarrollo de sitios web por lo general, contendrá como mínimo, las siguientes roles.

  • Diseñador Visual (ej., fuentes, colores, espacios, emoción, concepto y tema visual).
  • Diseñador UI/UX (ej., wireframes, especificar las interacciones y funcionalidad de la interfaz, estructurar información).
  • Desarrollador Front-End (ej., escribe código que se ejecuta en el cliente, navegador y/o dispositivo).
  • Desarrollador Back-End (ej., escribe código que se ejecuta en el servidor).

Un desarrollador front-end típicamente tiene que saber construir diseño de interfaces enfocado a tecnologías web.

Tip
